# VillageChef 🍳✨

VillageChef is a professional, single-page responsive website designed for culinary enthusiasts and food services. Built with a focus on clean UI/UX and mobile-first responsiveness.

## 🚀 Live Demo
Check out the live project here: [VillageChef Live](https://village-chef-website.web.app)

## 🛠️ Tech Stack
* **Frontend:** React.js (JSX)
* **Styling:** Bootstrap 5 & Custom CSS
* **Hosting:** Firebase Hosting

## ✨ Key Features
* Fully Responsive Design (Works on all devices)
* Modern & Clean Layout
* Single Page Application (SPA) structure
* Optimized Assets for fast loading
